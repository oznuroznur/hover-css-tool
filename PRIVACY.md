# Privacy Policy — CSS Inspector

_Last updated: 2026-07-17_

## English

CSS Inspector does not collect, store, transmit, or share any user data.

- **No data collection.** The extension has no analytics, telemetry, or
  tracking of any kind.
- **No network requests.** The extension never contacts any server. All
  processing happens locally in your browser.
- **Minimal access.** The extension only runs in a tab after you click its
  toolbar icon there (`activeTab` permission). It reads the styles of the
  element you select solely to display them to you. Nothing is persisted:
  when you close the tab or exit inspect mode, nothing remains.
- **Clipboard.** Text is written to your clipboard only when you press the
  Copy button, and only the code shown in the panel.

Because the extension processes everything locally and stores nothing, there
is no user data to access, export, or delete.

## Türkçe

CSS Inspector hiçbir kullanıcı verisi toplamaz, saklamaz, iletmez veya
paylaşmaz.

- **Veri toplama yok.** Uzantıda analitik, telemetri veya herhangi bir
  izleme bulunmaz.
- **Ağ isteği yok.** Uzantı hiçbir sunucuyla iletişim kurmaz; tüm işlem
  tarayıcınızda yerel olarak yapılır.
- **Asgari erişim.** Uzantı yalnızca araç çubuğu ikonuna tıkladığınız
  sekmede çalışır (`activeTab` izni). Seçtiğiniz elementin stillerini
  yalnızca size göstermek için okur. Hiçbir şey kalıcı olarak saklanmaz:
  sekmeyi kapattığınızda veya inspect modundan çıktığınızda geriye hiçbir
  şey kalmaz.
- **Pano.** Panoya yalnızca Kopyala butonuna bastığınızda ve yalnızca
  panelde görünen kod yazılır.

Her şey yerel işlendiği ve hiçbir şey saklanmadığı için erişilecek, dışa
aktarılacak veya silinecek bir kullanıcı verisi yoktur.
